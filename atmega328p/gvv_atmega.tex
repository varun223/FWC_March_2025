\begin{abstract}
%\boldmath
This manual shows how to use the ATMEGA328P microcontroller of the arduino for an embedded system.  The ATMEGA328P is first programmed using the arduino board and then taken out of the board and plugged into a breadboard along with a seven segment display.  Additional circuitry is assembled on the breadboard for powering the IC and using it for a decade counter.  Soldering this arrangement on a PCB results in a standalone embedded device.  
\end{abstract}
